# LORA TRAINER

This is a simple LORA trainer that can be used to train your Lora model on any dataset.

## Prequisites

- Python 3
- Images to be trained on

## Installation

```bash
chmod +x setup.sh
./setup.sh
```

## Usage

- Configure the config/appconfig.json file with your dataset paths and other parameters

```bash
python3 main.py
```
